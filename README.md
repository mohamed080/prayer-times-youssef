# 🕌 Prayer Times Youssef

A beautiful Islamic prayer times web app built with ❤️ as a **Sadaqah Jariyah** for the soul of **Youssef Mohamed Abdullah**.

---

## 🤍 About the Project

This project is dedicated as an ongoing charity (صدقة جارية) for the late **Youssef Mohamed Abdullah**, who passed away at the age of 18.

> “O Allah, forgive him, have mercy on him, and grant him the highest place in Paradise.”

The goal of this project is to provide:
- Accurate daily prayer times
- A calm Islamic user experience
- Reminders to make duaa for the deceased
- Continuous reward for everyone who benefits from it

---

## ✨ Features

- 🕌 Live prayer times (Aladhan API)
- 📍 City selection + GPS location detection
- ⏱ Countdown to next prayer
- 🔔 Notifications for prayer time alerts
- 🔊 Optional Azan sound playback
- 🌙 Dark / Light mode
- 📿 Digital Tasbeeh counter
- 🤲 Dedicated memorial section for duaa
- 📱 Fully responsive UI (mobile friendly)
- 🎨 Modern Islamic green gradient design

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Modern UI + Animations)
- Vanilla JavaScript (ES6)
- Aladhan API (Prayer times)
- OpenStreetMap (Location detection)
